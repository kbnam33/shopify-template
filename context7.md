Perfect! The Shopify CLI tool is exactly what you need for custom Liquid development. Here's what you should know:

When to Use Shopify CLI for Themes
The Shopify CLI for Themes is ideal when you're:

✅ Building custom Shopify themes with Liquid code
✅ Developing locally in VS Code
✅ Testing and debugging your theme files
✅ Exploring Liquid code interactively
Key Commands for Your Workflow
1. Start the Liquid REPL Console
Interactively test Liquid code in real-time:


shopify theme console

# With context for a specific page
shopify theme console --url /products/classic-leather-jacket
2. Profile Theme Performance
Optimize your Liquid rendering performance:


shopify theme profile

# Check specific pages
shopify theme profile --url /products/classic-leather-jacket
Additional Tools
You might also want to use Theme Check alongside CLI—it's a linter for themes that has a VS Code extension to catch issues as you code.
